Overview

The Luno Trading Bot is a professional-grade automated trading solution designed for the Luno cryptocurrency exchange. It provides advanced features for algorithmic trading, risk management, and portfolio monitoring.

Key Features

  • Multi-pair Support: Trade multiple cryptocurrency pairs simultaneously
  • Risk Management: Built-in stop-loss, position sizing, and portfolio protection
  • Real-time Monitoring: Live dashboard with charts and performance metrics
  • Simulation Mode: Test strategies without risking real money
  • Flexible Configuration: Customizable trading parameters and strategies

Architecture

The bot consists of several key components:

  • Trading Engine: Core logic for executing trades and managing positions
  • API Client: Interface to Luno exchange API
  • Risk Manager: Monitors and controls trading risks
  • Web Dashboard: Real-time monitoring interface
  • Data Storage: Trade history and performance tracking

Configuration

Environment Variables

Configure the bot using environment variables in a .env file: 

# Luno API Configuration
LUNO_API_KEY=your_api_key_here
LUNO_API_SECRET=your_api_secret_here

# Trading Configuration
TRADING_PAIR=XBTMYR
DRY_RUN=true
MAX_POSITION_SIZE=10
STOP_LOSS_PERCENTAGE=5
TAKE_PROFIT_PERCENTAGE=10

# Risk Management
MAX_DAILY_LOSS=1000
MAX_OPEN_POSITIONS=3
MIN_TRADE_AMOUNT=0.001

# Web Dashboard
WEB_PORT=5001
WEB_HOST=localhost
WEB_DEBUG=false

# Logging
LOG_LEVEL=INFO
LOG_FILE=logs/trading_bot.log

Configuration Parameters

Parameter Type Default Description
TRADING_PAIR string XBTMYR Cryptocurrency pair to trade
DRY_RUN boolean true Enable simulation mode
MAX_POSITION_SIZE number 10 Maximum position size as percentage of portfolio
STOP_LOSS_PERCENTAGE number 5 Stop loss threshold percentage
TAKE_PROFIT_PERCENTAGE number 10 Take profit threshold percentage

Trading Strategies

Built-in Strategies

The bot includes several pre-built trading strategies:

Mean Reversion

Buys when price is below moving average and sells when above. Good for ranging markets.

  • Uses 20-period moving average
  • RSI confirmation (oversold/overbought)
  • Volume confirmation

Momentum

Follows price trends and momentum. Suitable for trending markets.

  • MACD signal crossovers
  • Breakout detection
  • Trend confirmation

Grid Trading

Places buy and sell orders at regular intervals. Works well in sideways markets.

  • Configurable grid spacing
  • Dynamic grid adjustment
  • Profit taking at each level

Custom Strategies

You can implement custom trading strategies by extending the base strategy class: 

from src.bot.strategy import BaseStrategy

class MyCustomStrategy(BaseStrategy):
    def should_buy(self, market_data):
        # Implement your buy logic
        return True
    
    def should_sell(self, market_data, position):
        # Implement your sell logic
        return False

Risk Management

Position Sizing

The bot automatically calculates position sizes based on your risk parameters:

  • Fixed Percentage: Use a fixed percentage of portfolio per trade
  • Kelly Criterion: Optimal position sizing based on win rate and average returns
  • Volatility-based: Adjust position size based on market volatility

Stop Loss

Multiple stop loss mechanisms protect your capital:

  • Percentage Stop: Exit when loss exceeds specified percentage
  • ATR Stop: Dynamic stop based on Average True Range
  • Time Stop: Exit after specified time period

Portfolio Protection

  • Daily Loss Limit: Stop trading if daily losses exceed threshold
  • Maximum Positions: Limit number of concurrent positions
  • Correlation Limits: Avoid over-concentration in correlated assets

Examples

FAQ

Q: Is it safe to run the bot with real money?

A: Always start with simulation mode (DRY_RUN=true) to test your configuration. Only use real money after thorough testing and understanding the risks involved.

Q: What trading pairs are supported?

A: The bot supports all trading pairs available on Luno exchange, including XBTMYR, ETHMYR, XBTZAR, ETHZAR, and others.

Q: How much capital do I need to start?

A: There's no minimum requirement, but we recommend starting with at least the equivalent of $100-500 to allow for proper position sizing and risk management.

Q: Can I run multiple bots simultaneously?

A: Yes, you can run multiple instances with different configurations and trading pairs, but ensure you have sufficient API rate limits and capital allocation.

Q: How do I backup my trading data?

A: All trading data is stored in the logs/ directory. Regularly backup this folder to preserve your trading history and performance data.
