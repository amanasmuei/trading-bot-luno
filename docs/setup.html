Installation

1

Clone the Repository

Download the trading bot source code from GitHub:

git clone https://github.com/amanasmuei/trading-bot-luno.git
cd trading-bot-luno
2

Run Setup Script

Use the automated setup script to install dependencies:

# For automatic setup
python setup.py

# Or use the setup wizard for guided installation
python setup_wizard.py

The setup script will automatically create a virtual environment and install all required dependencies.

3

Alternative: Manual Installation

If you prefer manual installation:

# Create virtual environment
python -m venv trading_bot_env

# Activate virtual environment
# On Windows:
trading_bot_env\Scripts\activate
# On macOS/Linux:
source trading_bot_env/bin/activate

# Install dependencies
pip install -r requirements.txt

Configuration

Environment Variables

Create a .env file in the project root with your configuration: 

# Luno API Credentials
LUNO_API_KEY=your_api_key_here
LUNO_API_SECRET=your_api_secret_here

# Trading Configuration
TRADING_PAIR=XBTMYR
DRY_RUN=true
MAX_POSITION_SIZE=10
STOP_LOSS_PERCENTAGE=5
TAKE_PROFIT_PERCENTAGE=10

# Web Dashboard
WEB_PORT=5001
WEB_HOST=localhost

Security Warning: Never commit your .env file to version control. It contains sensitive API credentials.

Trading Parameters

TRADING_PAIR

The cryptocurrency pair to trade (e.g., XBTMYR, ETHMYR)

DRY_RUN

Set to true for simulation mode, false for live trading

MAX_POSITION_SIZE

Maximum percentage of portfolio to use per trade (1-100)

STOP_LOSS_PERCENTAGE

Stop loss threshold as percentage (1-50)

1. Create Luno Account

If you don't have a Luno account, create one here.

2. Generate API Keys

3. Test API Connection

Verify your API credentials work:

python -m tests.test_api_credentials

Running the Bot

Quick Start

Use the launcher for easy startup:

python launcher.py

Manual Start

Run components separately:

# Start the trading bot
python scripts/run_bot.py

# In another terminal, start the web dashboard
python src/web/dashboard.py

Docker

Run using Docker:

# Build and run with Docker Compose
docker-compose up -d

# Or build manually
docker build -t luno-trading-bot .
docker run -d --env-file .env -p 5001:5001 luno-trading-bot

Troubleshooting

API Connection Failed

  • Verify API credentials in .env file
  • Check API permissions on Luno
  • Ensure internet connection is stable

Module Import Errors

  • Activate virtual environment
  • Reinstall dependencies: pip install -r requirements.txt
  • Check Python version (3.8+ required)

Web Dashboard Not Loading

  • Check if port 5001 is available
  • Verify WEB_HOST and WEB_PORT in .env
  • Check firewall settings

Trading Not Working

  • Ensure DRY_RUN is set correctly
  • Check account balance
  • Verify trading pair is supported
